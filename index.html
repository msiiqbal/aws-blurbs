CW Agent

Troubleshooting the CloudWatch agent - CloudWatch agent files and locations - https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/troubleshooting-CloudWatch-Agent.html#CloudWatch-Agent-files-and-locations


a) config.json file  - Agent configuration file. 
Linux : /opt/aws/amazon-cloudwatch-agent/bin/config.json
Windows : & $Env:ProgramFiles\Amazon\AmazonCloudWatchAgent\config.json

b) amazon-cloudwatch-agent.log file - The Log file the agent writes to.
Linux : /opt/aws/amazon-cloudwatch-agent/logs/amazon-cloudwatch-agent.log
Windows : & $Env:ProgramData\Amazon\AmazonCloudWatchAgent\Logs\amazon-cloudwatch-agent.log

c) configuration-validation.log file - Agent configuration validation file.
Linux : /opt/aws/amazon-cloudwatch-agent/logs/configuration-validation.log
Windows : & $Env:ProgramData\Amazon\AmazonCloudWatchAgent\Logs\configuration-validation.log

d) TOML Files.
  i) amazon-cloudwatch-agent.toml : 
  Linux : /opt/aws/amazon-cloudwatch-agent/etc/amazon-cloudwatch-agent.toml
  Windows : & $Env:ProgramData\Amazon\AmazonCloudWatchAgent\amazon-cloudwatch-agent.toml

  ii) common-config.toml : 
  Linux : /opt/aws/amazon-cloudwatch-agent/etc/common-config.toml
  Windows : & $Env:ProgramData\Amazon\AmazonCloudWatchAgent\common-config.toml

e) file_config.json :
Linux : /opt/aws/amazon-cloudwatch-agent/etc/amazon-cloudwatch-agent.d/file_config.json
Windows : C:\ProgramData\Amazon\AmazonCloudWatchAgent\Configs\file_config.json

f) YAML file which is conversion of json to YAML
Linux : /opt/aws/amazon-cloudwatch-agent/etc/amazon-cloudwatch-agent.yaml
Windows : & $Env:ProgramData\Amazon\AmazonCloudWatchAgent\amazon-cloudwatch-agent.yaml


Other commands 

CW agent Configuration
1) Linux : sudo /opt/aws/amazon-cloudwatch-agent/bin/amazon-cloudwatch-agent-config-wizard
2) Windows : cd "C:\Program Files\Amazon\AmazonCloudWatchAgent" amazon-cloudwatch-agent-config-wizard.exe

CW Agent strat cmd
1) Linux :  sudo /opt/aws/amazon-cloudwatch-agent/bin/amazon-cloudwatch-agent-ctl -a fetch-config -m ec2 -c file:configuration-file-path -s 
  example : sudo /opt/aws/amazon-cloudwatch-agent/bin/amazon-cloudwatch-agent-ctl -a fetch-config -m ec2 -c file:/opt/aws/amazon-cloudwatch-agent/bin/config.json -s
2) Windows : & $Env:ProgramFiles\Amazon\AmazonCloudWatchAgent\amazon-cloudwatch-agent-ctl.ps1 -a fetch-config -m ec2 -c file:configuration-file-path -s 
  ---> & $Env:ProgramFiles\Amazon\AmazonCloudWatchAgent\amazon-cloudwatch-agent-ctl.ps1 -a fetch-config -m ec2 -c file:'C:\Program Files\Amazon\AmazonCloudWatchAgent\config.json' -s

CW Agent Stop Cmd
1) Linux : sudo /opt/aws/amazon-cloudwatch-agent/bin/amazon-cloudwatch-agent-ctl -m ec2 -a stop
2) Windows : & $Env:ProgramFiles\Amazon\AmazonCloudWatchAgent\amazon-cloudwatch-agent-ctl.ps1 -m ec2 -a stop

CW Agent Status Check Cmd
1) Linux : sudo /opt/aws/amazon-cloudwatch-agent/bin/amazon-cloudwatch-agent-ctl -m ec2 -a status
2) Windows : & $Env:ProgramFiles\Amazon\AmazonCloudWatchAgent\amazon-cloudwatch-agent-ctl.ps1 -m ec2 -a status
